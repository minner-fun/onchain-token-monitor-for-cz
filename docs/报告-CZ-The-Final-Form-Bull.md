### 资金上游 = four.meme 发射台 + 一个币安供血的「BNB 水龙头」(不是 CZ 的主谋)

顺着钱往上追,CZ 的上游是**公共基础设施**,不是一个封闭的私人工厂。这一点我们一开始判断过了头,这里更正。

**① 发射台 = four.meme。** CZ(以及那些 `…4444` 币)大多在 **[four.meme](https://four.meme)** 上发——一个 BSC 的公开发币平台(债券曲线,类似 pump.fun)。任何人付费就能发,`…4444` 是它磨的默认靓号,所以全网约六成新土狗都长这样。**four.meme 只卖铲子,不管你的币死活;发完得自己拉、自己护盘。**

**② 资金上游 = 一个币安供血的「BNB 水龙头」`0xe2fc…`。** 追它的完整资金流:
- **进:** **141,862 BNB(≈$8,200 万)几乎全部来自一个币安热钱包**;非 CEX 进账只有 0.03 BNB(尘埃);
- **出:** **138,100 BNB 单向分发给 63,602 个不同地址**;
- **利润不回流**——所以它**不是收割利润的金库**,而是一个**给六万多地址供 BNB 的分发水龙头**。

- ✅ **实证**:`0xe2fc` 收币安 ~14.2 万 BNB、单向分发给 **63,602 个地址**、利润不回流(非 CEX 进账仅 0.03 BNB)。
- ✅ **实证(定性,见下 ③)**:它**不是 four.meme 官方地址**——与 four.meme 官方合约 0 交互、从没在 four.meme 上发过币 / 代发过币,下游里只有 0.27% 是 four.meme 发币人。
- 🔶 **推断**:它是**共享的资金基础设施(服务 / 水龙头),不是单一操盘方的马甲农场**——一个操盘拉 CZ 一个盘,不需要几万个钱包。
- ⛔ **不可知**:它究竟是"币安关联的批量提现 / 代付通道"还是"第三方 OTC / 代付服务"(背后运营方的确切身份)——链上无法确证。

**③ 定性:它是不是 four.meme 官方?——不是(✅ 实证排除)。** 再追一跳 `0xe2fc` 与 four.meme 官方合约(`TokenManager2` `0x5c952063…9550762b`、v1 `0xEC4549…5233bFbC`)的关系,四条测试全指向"无关":

### 拉盘怎么做出来的:一套「执行器 / 库存」分离的做市舰队（✅ 实证）

深挖那些在链上"高位卖出"的地址,发现一个关键事实——**它们本身一枚 CZ 都不留,是执行 bot(签名下单),不是持有人**:

- 那批"赢家"(`0xb71490c` 卖 $442k、`0x4337…` 家族各卖 $100k+)**收到的 CZ 转账也是 0**——它们是**卖出执行器**,卖的是别处路由来的库存,自己不持仓。
